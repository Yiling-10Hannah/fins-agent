# Presentation Rules

Use these rules for student course presentations and PowerPoint decks.
Legacy Beamer decks use these same presentation principles only when
explicitly requested.

## Core Principles

- One idea per slide.
- Titles should state assertions, not labels.
- Slides support the spoken argument; they are not the report.
- Remove any element that does not help the audience see the point.

## Structure

- Start with the research question and why it matters.
- Show data scope, source, and sample period early.
- Put the main results in assertion-title slides.
- Include limitations or robustness only when they change interpretation.
- End with the one takeaway the audience should remember.

## Visuals

- Prefer direct labels over legends.
- Highlight only the comparison the audience should notice.
- Avoid full regression tables on slides; extract the rows that matter.
- Keep body text large enough to read in a classroom.
- Use figures generated through `fintools.figures` where possible.
