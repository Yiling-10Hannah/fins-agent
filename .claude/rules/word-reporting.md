# Word Reporting Rules

Use these rules for student coursework reports written in Microsoft Word.
Word `.docx` is the default report source.

## Structure

- Save report files under `report/`, normally `report/report.docx`.
- Use built-in Word styles: Title, Heading 1, Heading 2, Heading 3, Normal,
  Caption, and Bibliography.
- Do not fake headings with bold Normal text.
- Standard sections are Abstract, Introduction, Data, Methodology, Results,
  Conclusion, References, and Appendix unless the assignment says otherwise.
- Use short, descriptive Heading 1 titles.

## Figures And Tables

- Use `fintools.figures` for Word/A4-ready plots before inserting images.
- Insert figures as inline images unless a specific layout requires wrapping.
- Use Word captions for figures and tables.
- Captions must define variables, units, sample period, and the main object.
- Use Word cross-references; do not type figure or table numbers manually.
- Use Word tables for short final tables.
- Put large raw outputs in an appendix or separate results file.
- Report 2 to 3 significant digits in final tables.

## Citations

- Use Word's References tab as the default citation workflow.
- Verify each citation before adding it.
- Do not invent author, year, title, journal, volume, or page metadata.
- Use `[HUMAN EDIT REQUIRED: verify citation]` when a source cannot be
  confirmed.
- Update the bibliography after changing citations.

## Final Checks

- Accept or reject tracked changes.
- Remove resolved comments.
- Update all fields: table of contents, captions, cross-references, and
  bibliography.
- Run Word's Accessibility Checker.
- Export the final PDF from Word after fields are updated.
