# Citation Verification Rules

Never invent citation metadata. Verify citations before adding them to a Word
source list, bibliography, or legacy BibTeX file.

## Existing Citations

- Confirm the cited source exists in the current report source list or `.bib`
  file.
- Confirm the cited source supports the claim being made.
- Check author names, year, title, venue, and publication status.

## New Citations

- Search the current project sources first.
- If the source is not already present, verify it with a trusted lookup before
  adding it.
- If a field is not confirmed, leave it out or mark it as unconfirmed.
- If the source cannot be verified, do not cite it.

## Field Rules

- Authors: every author must be confirmed.
- Title: use the exact title.
- Year: match the verified source.
- Journal, volume, issue, pages: include only if confirmed.
- Working papers and forthcoming articles need explicit publication-status
  notes.

## Legacy BibTeX

- Use `@article` for published articles and `@misc` or `@unpublished` for
  working papers.
- Protect proper nouns in titles with braces.
- Do not fill missing journal, volume, or page fields from memory.
