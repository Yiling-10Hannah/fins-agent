# Banned Words And Phrases

Use this list when drafting, editing, and proofreading coursework reports.
Prefer concrete finance language over generic AI-sounding prose.

## Hard Bans

| Avoid | Prefer |
|-------|--------|
| delve, delve into | examine, study, analyze |
| crucial, paramount | important, central |
| comprehensive | detailed, complete, or a specific scope |
| multifaceted | complex, varied |
| utilize | use |
| leverage, when it means use | use, employ |
| facilitate | allow, enable, help |
| myriad, plethora | many, several |
| noteworthy | state the fact directly |
| it is important to note that | delete |
| it should be noted that | delete |
| in this regard | delete or restructure |
| in order to | to |
| due to the fact that | because |
| the vast majority of | most |
| in the context of | in, for, when |
| with respect to | for, about, regarding |
| in terms of | restructure |
| as such | therefore, or delete |
| showcasing, showcase | showing, show |
| pivotal | central |
| intricate | complex |
| meticulous, meticulously | careful, carefully |
| illuminates, unveil | shows, reveals |
| realm, landscape | field, market, setting |
| aligns with, in line with | matches, is consistent with |
| in other words | rewrite the first sentence |
| of course, obviously | delete |
| aforementioned | this plus a noun |
| shed light on | reveal, show |
| innovative | new, if genuinely new |
| not only ... but also | split into two direct sentences |
| plays a role in | name the mechanism |
| taken together | state the conclusion directly |

## Soft Bans

Use these sparingly, and replace them with numbers where possible:

- significant, significantly
- novel
- robust outside "robustness checks"
- key
- critical, critically
- highlights
- insights
- compelling
- ultimately
- somewhat, quite, very, rather, arguably, perhaps
- especially, particularly
- a variety of
- in general
- Indeed, as a sentence opener

## Structural Patterns To Avoid

- "Together, these results..." as a repeated paragraph opener.
- "This finding..." more than once per report.
- "In this section, we..." unless the sentence names a specific question.
- Consecutive paragraphs that start with the same grammar pattern.
- Closing paragraphs that merely restate what the section already said.
- Gerund-phrase opener density, such as repeated "Using..." sentences.
