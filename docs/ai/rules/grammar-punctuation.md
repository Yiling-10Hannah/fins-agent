# Grammar And Punctuation

Use the Chicago Manual of Style as the coursework baseline unless an
assignment says otherwise.

## Core Rules

- Use the serial comma: "data, methods, and results".
- Use "that" for restrictive clauses and "which" for nonrestrictive clauses.
- Use semicolons only between independent clauses or in complex lists.
- Use colons only after an independent clause.
- Hyphenate compound modifiers before nouns: "well-known result".
- Do not hyphenate adverbs ending in `-ly`: "statistically significant".
- Use digits for 10 and above, units, percentages, statistics, and table
  quantities.
- Spell out one through nine in ordinary prose.
- Do not start a sentence with a variable or numeral.
- Avoid comma splices; split the sentence or use a semicolon.
- Use double quotation marks for terms and direct quotations.

## Word Reports

- Use Word equation editor for equations.
- Punctuate displayed equations as part of the sentence.
- Use Word cross-references rather than typed figure, table, or equation
  numbers.

## Legacy LaTeX

- Use LaTeX quotation marks correctly.
- Use `--` only for ranges and compound modifiers in LaTeX source.
- Do not use `---` as a prose dash in reports; rewrite the sentence.
