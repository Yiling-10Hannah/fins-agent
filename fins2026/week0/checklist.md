# Setup Checklist

Copy this list and check off each item as you complete it.

## Before Class

- [ ] Git installed ([Windows](https://git-scm.com/download/win) |
      [Mac](https://git-scm.com/download/mac))
- [ ] GitHub account created and email verified
      (required before the OpenCode in PyCharm path)
- [ ] Editor installed:
      [PyCharm](https://www.jetbrains.com/pycharm/download/) or
      [VS Code](https://code.visualstudio.com/)
- [ ] Repo cloned: `git clone <URL>`
- [ ] PyCharm opened the cloned `fins-agent` folder as one project

## Setup

- [ ] Chose an AI path from
      [start-here.md](../../docs/ai/start-here.md)
      or followed [QUICKSTART.md](../../QUICKSTART.md) manually
- [ ] PyCharm terminal is at the repo root before setup commands run
- [ ] Ran the onboarding workflow in your chosen tool
- [ ] The repo interpreter run of `tools/setup_student.py` shows all checkmarks
- [ ] Editor can find the Python interpreter
      (PyCharm: auto-detects `.venv/`; VS Code: select the repo
      interpreter)

## Ready to Go

- [ ] Opened `fins2026/week0/` and read this checklist
- [ ] Read [ai-workflow.md](ai-workflow.md)
- [ ] Know where to find
      [PyCharm setup](../../docs/setup/pycharm.md),
      [AI support matrix](../../docs/setup/ai-support-matrix.md), and
      [AI troubleshooting](../../docs/setup/ai-troubleshooting.md)
- [ ] Tried asking the AI a question about the repo
      (for example, "What packages are installed?" or
      "Show me the project structure")
- [ ] Know where weekly exercises live (`fins2026/weekN/`)
- [ ] Know where project work goes (`projects/`)
